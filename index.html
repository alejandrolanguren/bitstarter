CoderDojo is a great collaboration project to help kids to learn the magnificent art of coding, this is a well established method to entice kids and parents to teach and learn how to code in a fun and cooperative environment. The SkyCoderDojo project want to improve the paradigm of CoderDojo, bringing this project finally to a neutral collaborative environment. SkyCoderDojo has the vision of extend this great collaborative learning process to a new hall in the web. Currently the philosophy of CoderDojo requires the creation of Dojos where parents and children neighbors collaborate to learn the basics of coding. This is a great process, although no without some challenges; a recommended one parent-one children ratio and a physical space to meet in short distance to all the Dojo members. So, how about potential parent/child teams who want to participate but for some reason are far apart? or a children that has a mentor but with the same distance constraints?

Well here's where SkyCoderDojo comes to the rescue. offering a mean to allow this experience in a common space in the web. SkyCoderDojo, using Google+ Hangouts in a controlled interface will allow the collaboration in virtual Dojos either formed in a closed environment (well-known participants), or a more interesting open environment where Dojos will be formed based in the participants availability and common objective for the Dojo session, and maintaining in the process the controlled ratio of one parent - one child. Allowing to all kids and parents who want to live and share this experience, regardless of distance constraints. Help SkyCoderDojo to extend the great CoderDojo project a step further, to include a new level of collaboration.
